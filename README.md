Code of Problem Solving and Computer Programming (PSCP/PSIT).

## Prefix and Suffix
#### wk{x}
- Week {x}
- To show where this problem comes from which week.

#### wk{x}m{y}
- Week {x}
- Midterm {y}
- To show where this problem comes from which week.
- Part of midterm exams.

#### (R)
- Recommend
- This problem will also a part of exams.

#### (U)
- Unfinished or unsolve problem
